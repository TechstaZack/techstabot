TechstaBot Commands



General Commands

/info

Gives info on the bot, as well as a server and youtube invite.

/suggest

As long as there is a suggest channel set, users can use this command to suggest things for the server.

/announce

Announces a message or embed in any channel.

/techstabot-channels

Set up, Remove or automatically create server suggest and report channels.

/uesrdm

Dm any user, as long as they dont have the bot blocked.

/stick & /unstick

Create a sticky embed that will repeat how often you tell it to.

/ReactionRole

Create a reaction role on any message!

/autoreact

Chose up to 6 emojis for the bot to automatically react to whenever a message is sent in a channel.

/message_requirements

Set message requirements for a channel, or block things entirely, shuch as requiring files but blocking links.

/broadcast

Broadcasting is a feature to create a channel that will connect to a specific broadcast, which allows members from other servers to communicate to eachother. There are millions of different codes youc an use for a broadcast ID, or you can connect to a public one (default, gtag, pokemon or gtagmod)!

And there is more to come! The bot is still experimental, but that doesnt mean its not useful. Use the button below to view custom commands!
